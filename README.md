## Packages

### 1. Pangu

#### Start up

Install jekyll and jekyll plugins. And run `yarn start` to develop.

```shell
> gem install jekyll
> gem install jekyll-paginate
> yarn start
```

View [http://127.0.0.1:4000/](http://127.0.0.1:4000/).

### 2. shennong

#### Start up

A create-react-app project.

